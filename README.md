# COCO_dataset

## Common Objects in Context

COCO is an open source project, with the objective of provide a very useful dataset to Computer Vision projects.

I've personally used it for two of my favorite projects and it has worked pretty well, so I've designed this repository to optimize the usage of this dataset in future projects.

**Source Link**: <https://cocodataset.org/#download>

### Images

- 2017 Train images [118K/18GB] (118,287 items)
- 2017 Val images [5K/1GB] (5,000 items)
- 2017 Test images [41K/6GB] (40,670 items)
- 2017 Unlabeled images [123K/19GB] (123,403 items)

Total count: 287,360

### Annotations

- 2017 Train/Val annotations [241MB]
